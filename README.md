# Coolzy Sneakers

Welcome to the Coolzy Sneakers project! This is an online sneaker store that showcases a variety of popular sneaker brands, allowing users to browse products and make purchases securely.

## Features

- **Product Showcase**: Displays various sneakers from brands like Jordan, Fila, Nike, and Adidas with pricing and discounts.
- **Responsive Design**: The website is optimized for both desktop and mobile devices.
- **Secure Payment Form**: Users can input their payment information securely.
- **Navigation**: Easy navigation to different sections of the site.

## Technologies Used

- HTML
- CSS
- JavaScript (for any future interactive features)
- Font Awesome (for icons)
